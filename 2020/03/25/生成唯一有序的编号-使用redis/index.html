生成唯一有序的编号-使用redis
发表于 2020-03-25| 更新于 2020-03-25
阅读量:

    项目中有种场景需要生成唯一、有序、自增编号，因为项目是分布式部署，这使得编号的生成有点麻烦。最后在同事的建议下使用Redis的Hincrby方法来简洁有效的解决这种问题。

java
  • Redis Hincrby 命令用于为哈希表中的字段值加上指定增量值。增量值可以为负数；相当于对指定字段进行减法操作。
